🌱 Plant Health Monitoring System
📌 Overview The Plant Health Monitoring System is an IoT and AI-powered project that integrates hardware, web technologies, and deep learning to monitor plant health in real time. It provides a responsive web interface for users to track plant conditions and leverages machine learning to detect leaf damage. 🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript, Bootstrap (for responsive design)
Microcontroller: ESP8266 (for sensor data collection and IoT connectivity)
Backend: Flask framework (to connect the web interface with the deep learning model)
AI Model: Deep learning model for leaf damage detection
Website Integration: Seamless connection between microcontroller, backend, and AI model ⚙️ How It Works
The ESP8266 microcontroller monitors plant health parameters and leaf conditions.
If leaf damage is suspected, the system captures data/images and sends them to the Flask backend.
The deep learning model analyzes the leaf image to determine whether it is healthy or damaged.
Results are displayed on the web interface, allowing users to view plant health status in real time. 🚀 Key Features
Real-time monitoring of plant health using IoT sensors
AI-powered leaf damage detection
User-friendly web interface for visualization
Scalable integration of hardware and software components
